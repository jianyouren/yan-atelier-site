## 📋 Cross-cutting patterns (B-level)

- **B1 · Every section anatomy is identical** (overline → display H2 → italic-EN sub → N-col grid). Vary register per section — small caps / lowercase italic / no overline. Real editorial brands (T Magazine, Apartamento) vary per section.
- **B2 · Chrome compensating for missing photography** — vignettes, scrim overlays, zoom-on-load, italic photo credits, AI-render badges all signal the image isn't doing the work. Photography gap is the REAL "太简陋" cause.
- **B3 · One display face × every H2** — Italiana at H1/H2/section-title/nav-brand/product-name/footer-h3/maker-signature loses signature value. Wear it ONCE.
- **B4 · Symmetric vertical padding everywhere** (`--pad-section` on 11 sections) — premium agencies tune bottom-padding slightly larger than top per section.
- **B5 · Single `--silver` / `--accent` token used universally** — when accent is on overlines + prices + hovers + drop cap + dividers, it loses informational value. Build 2-tier accent system: commerce signals only vs chrome accents.

## Lens 4 · soft-skill

**Diagnosis:** chrome production-value deficit is real (friend's instinct correct), but cause is **draftsmanship density at hairline scale** — the gap separating Pentagram from Canva. NOT addressable by maximalism.

**Typography:** Italiana display is single biggest cheapness signal (free Google revival, no italic, no optical sizing, brittle hairlines, wedding-invitation template at hero scale) · tracking is wrong by sign (display tracked open, body tight or negative — premium does the opposite) · overline tracking uniform 0.22-0.32em across 50+ rules · Cormorant 400 body + line-height 1.7 = Etsy serif look (use 500 + line-height 1.55) · missing tabular-nums + hanging-punctuation on surfaces that need them · CN+Latin pair uneven (Italiana has no CJK fallback)

**Spacing:** section padding inconsistent (5 different rhythms in one scroll — silent killer of agency feel) · value-section gutter too tight (~2rem vs WWAKE 5rem+) · hero CTA gap 1.5rem is Bootstrap-default · product-grid offset uniform (stagger varies)

**Cards/surfaces:** `.testimonial-card` + `.ugc-card` carry `border: 1px solid` AND `box-shadow: 0 2px 12px rgba(22,19,16,0.04)` = textbook Bootstrap "Material lite" stack — highest-confidence cheapness signal in the file · `.value-icon` emoji-style dingbats render inconsistently across OS · `.press-quote-card` uses Medium pull-quote default (border-left: 3px) · no nested surfaces anywhere (missing "double-bezel / Doppelrand" print-monograph signal)

**Motion:** `transition: ... ease` 40+ times (banned curve per spec — sed-pass `ease` → `cubic-bezier(0.16,1,0.3,1)` worth 30% perceived-quality lift) · `.btn` has no hover micro-motion / press feedback / trailing icon · dual-door hover only scales bg image (add type lift + arrow lift + scrim concert) · `.reveal-on-scroll` system set up but evidently never applied (zero `class="reveal-on-scroll"` in HTML)

**3 specific moves that close the gap WITHOUT ornament/gold/glow:**

- **Move A (typography):** Replace Italiana with GT Sectra Display / PP Editorial New / Migra. Tighten display tracking to -0.022em on all `font-size ≥ 2rem`. Body Cormorant weight 400 → 500, line-height 1.7 → 1.55. CN body weight 400 → 500.
- **Move B (motion):** Sed `ease` → `cubic-bezier(0.16, 1, 0.3, 1)` everywhere. Add to `.btn-primary, .btn-ghost, .featured-link, .plique-link, .atelier-cta, .about-craft-cta`: `:hover { transform: translateY(-1px); box-shadow: 0 12px 32px -16px rgba(22,19,16,0.45); } :active { transform: translateY(0) scale(0.985); transition-duration: 120ms; }`. Wire IntersectionObserver to actually apply reveal-on-scroll to hero/door/featured/maker/plique/promise sections with `transition-delay: calc(var(--i) * 80ms)`.
- **Move C (surfaces):** Kill border + Material-lite shadow on `.testimonial-card / .ugc-card`. Replace `✦ ◈ ⚒` with inline 24×24 SVG (1.25px stroke, currentColor, geometric — ring/square-in-square/triangle). Add `box-shadow: inset 0 0 0 1px rgba(251,247,238,0.08)` (light variant) or `inset 0 0 0 1px rgba(22,19,16,0.06)` (dark variant) to `.hero-img, .plique-img, .maker-img, .atelier-img, .pd-gallery, .product-img, .door-bg` — single hairline inset = "image is mounted in a paper window-mat" = print monograph signal.

**What NOT to do (explicitly resisted):** no gold accent re-introduction · no radial glow on promise-section (already removed) · no expansion of dingbat ornaments · no hero image scaling up · no autoplay video on hero · no parallax · no marquee revival · no vertical CN ornament return.
