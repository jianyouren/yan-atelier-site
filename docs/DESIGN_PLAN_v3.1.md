## 六、Signature 设计系统

### 6.1 窗口持光 (Window to Light)

**核心概念**: 用户的光标/触摸 = 手持一扇窗，照亮珐琅

#### 桌面端：光标跟随

```css
.light-reveal {
  position: relative;
  overflow: hidden;
}

.light-reveal::after {
  content: '';
  position: absolute;
  inset: 0;
  background: radial-gradient(
    420px circle at var(--mx, 50%) var(--my, 50%),
    rgba(255, 230, 180, 0.55) 0%,
    rgba(255, 220, 160, 0.25) 40%,
    transparent 75%
  );
  pointer-events: none;
  mix-blend-mode: soft-light;
  opacity: 0;
  transition: opacity 0.3s ease;
}

.light-reveal:hover::after {
  opacity: 1;
}

@media (prefers-reduced-motion: reduce) {
  .light-reveal::after {
    display: none;
  }
}
```

#### 移动端：自动漂移

```css
@media (hover: none) {
  .light-reveal::after {
    --mx: 30%;
    --my: 60%;
    opacity: 0.7;
    animation: lightDrift 8s ease-in-out infinite alternate;
  }

  @keyframes lightDrift {
    0%   { --mx: 30%; --my: 60%; }
    50%  { --mx: 70%; --my: 40%; }
    100% { --mx: 50%; --my: 70%; }
  }
}
```

### 6.2 品牌色彩系统

```css
:root {
  /* 核心色 */
  --ivory: #FBF7EE;
  --ink: #161310;
  --ink-soft: #2A2622;

  /* Signature 色 */
  --kiln-amber: #C4572A;     /* 窑火色 */
  --enamel-blue: #2A5C6E;    /* 珐琅蓝 */
  --sterling: #A8A19A;       /* 银丝灰 */

  /* 辅助色 (修复对比度) */
  --hush: rgba(22, 19, 16, 0.65);  /* 从 0.55 → 0.65 */

  /* 珐琅渐变 */
  --enamel-glow: linear-gradient(
    135deg,
    rgba(196, 87, 42, 0.15) 0%,
    rgba(42, 92, 110, 0.12) 50%,
    rgba(200, 180, 150, 0.08) 100%
  );
}
```
