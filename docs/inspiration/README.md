# docs/inspiration/

**Mood board + 设计参考素材。永远不上线。**

这个目录里的图都是 YÀN Atelier 设计过程中的灵感参考 — 来源是 Pinterest / 别家品牌官网截图 / Misho / Wing Yau / Yvmin / Mejuri / Sophie Bille Brahe 等 indie 品牌的视觉范式。

## 硬规则

- ❌ **不要把这里的图复制到 `/images/`** — 那是部署到 yan-atelier.com 的目录
- ❌ **不要在 `index.html` 里引用这些路径** — Cloudflare Pages 会把这些图一起部署,等于在自己站上展示别家产品
- ❌ **不要让 AutoEdit / WeChat 桥把这些当 SKU 参考** — 它们不是 YÀN 的产品
- ✅ 可以参考 mood / 构图 / 配色给设计师 / Claude 看,告诉他们"我要这种感觉"
- ✅ 可以喂给 image-gen 作 `style_ref` (注意只取审美,不取产品)
