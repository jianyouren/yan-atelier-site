## Listing-wide rules (apply to all 3)

### SEO keyword priorities (Etsy algorithm)
1. **Title front-load:** "Plique-à-jour" / "Mineral enamel" / "Wire-inlay" — these are the high-intent + low-supply search terms
2. **Tags must mix:** craft term + material + form + gift context + maker style
3. **Description first 160 chars:** load with high-intent keywords (Etsy uses these for relevance + Google snippet)

### Trust badges to enable
- [ ] Etsy's "handmade" toggle (always TRUE)
- [ ] "Made to order" badge (toggle on)
- [ ] Star Seller eligibility (requires 95%+ 5-star, on-time shipping, fast replies)
- [ ] Gift wrapping option (separate listing $5-10)

### Reviews bootstrap (first 30 days)
- Send to 3-5 personal contacts at full price (Etsy detects family/friend gifting through IP overlap, OK at low volume)
- Each one leaves an honest review with photo
- 5 reviews unlocks search visibility tier

### Cross-link to DTC site
- In every photo gallery, last image = "Full catalog at yan-atelier.com" with QR code
- This drives Etsy → DTC migration over 6-12 months (Etsy takes 6.5% + payment processing; DTC keeps 100% minus card fees)

### Pricing rule
- Etsy minimum margin must absorb: 6.5% transaction fee + $0.20 listing fee + 3-4% payment processing + international shipping subsidy if free-shipping offered
- Approximate net to YÀN at $199 listing = $169-174 after fees (excluding shipping if charged separately)
- This is OK for entry-level Phase 1; DTC main pieces should never go through Etsy

## What NOT to put on Etsy

- Bespoke commissions ($899+) — Etsy buyers don't commission, they buy off-the-shelf
- The premium Iris Lattice / Long-life Lock pieces ($428+) — these need PDP storytelling Etsy doesn't allow
- Hu Yan portrait or atelier scene as standalone product — keep these on DTC for brand
- Anything with the word "limited edition" — Etsy algorithm punishes scarcity claims
